Authentication

GET /api/v1/login
  • Description

    Authenticate a user using their username and password.

  • Params

    Body (email, password)

  • Response (Status code 200) 

                                        
                                        {
                                            "data": {
                                                "refresh": "refresh-token",
                                                "token": "session-token"
                                            },
                                            "message": "Ok",
                                            "success": true
                                        }

GET /api/v1/refresh-token
  • Description

    Generate a new session token by validating an active refresh token.

  • Params

    Header (Authorization: Refresh token)

  • Response (Status code 200) 

                                        
                                        {
                                            "data": "session-token",
                                            "message": "Ok",
                                            "success": true
                                        }
